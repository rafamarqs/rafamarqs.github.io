Commit com quase todos os arquivos

19.12.26

Se não quer adicionar todos os arquivos ao próximo commit, a resposta pode ser apenas não adicionar esses arquivos. Mas e se já tiver adicionado? Para isso temos o comando ‘git reset HEAD’.

Basta usar esse comando seguido pelo nome do arquivo que deseja remover da área de preparação.

Ex: git reset HEAD arquivo.txt

Lembrando que se o nome do arquivo tiver espaço, use entre aspas.

Ex: git reset HEAD ‘arquivo 2.txt’
