Já escolheu um Sistema Operacional?

19.07.14

Sempre gostamos de ver todas as opções antes de escolher, mas nem sempre sabemos que elas existem. Se já comprou um smartphone ou tablet ou pc, talvez tenha feito pesquisas para saber como fazer uma boa escolha.

Já escolheu um Sistema Operacional?

A nossa realidade é interessante nesse assunto, certa vez apresentei uma distro pra uma amiga e ela disse que não sabia que existia outro sistemas além do Windows. E na verdade eu mesmo quando comecei a conhecer um computador, só fui apresentado ao Windows, não tinha opções.

Windows

O Windows é o sistema que já vem na maioria dos PCs que compramos, então é normal ser conhecido, mas não conhecer outras opções é algo bem chato de se pensar, afinal, ter escolha é um direito, ou não?

Mac OS

O Sistema da Maçã já é mais isolado, se desejar usar, tem que deixar de lado diversas opções em equipamento. Sua liberdade de escolha é bem limitada. Mas se consegue se sentir bem dentro desses limites, que bom, não é pra todos, mas se está bem, tá bom.

Distros Linux

Outra opção, na verdade, outras opções, são as distros de código aberto, são milhares de opções, são tantas que na chega a ser um problema. Qual escolher? Qual o melhor? Perguntas simples e respostas complexas.
