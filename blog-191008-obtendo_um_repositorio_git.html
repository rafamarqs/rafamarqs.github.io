Obtendo um repositório Git

19.10.08

Chegou o momento de ter um repositório Git, podemos fazer isso facilmente de duas formas. Podemos iniciar um repositório, preparando nossos arquivos. Podemos também clonar um repositório que existe em outro lugar.

Iniciando um repositório Git

Se você já tem algum diretório (pasta) que quer ter um controle de versão, primeiro precisamos ir até esse diretório. Dentro dessa pasta podemos clicar com o botão direito e abrir o Git Bash ou o Terminal.

E digite:

git init

Pronto! Iniciamos um repositório Git, mas calma, ainda não está sendo rastreado. Para começar precisamos adicionar os arquivos que queremos, como estamos começando, vamos adicionar todos.

Digite:

git add *

isso adiciona todos os arquivos na area de preparação do Git, ainda precisamos fazer um um commit que é adicionar os arquivos no formato em que estão na area de preparação.

Digite:

git commit -m ‘iniciando controle de versão’

Pronto! Já temos tudo o que precisamos até o momento.

Clonando um repositório Git

Podemos obter uma cópia completa de um repositório Git que existe em outro lugar.

Por ser uma cópia completa, é possível que a sua cópia possa depois servir como cópia de segurança e ser usada para recuperar os dados caso aconteça algum problema no servidor.

Para clonar usamos o comando git clone, vamos começar com um repositório bem popular, o bootstrap, que é um framework web

Vá para o diretório onde deseja que esse repositório esteja.

Digite:

git clone https://github.com/twbs/bootstrap.git

Agora você possui um repositório completo do bootstrap.
